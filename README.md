# 🎬 Inclusive AI — Расшифровка видео с помощью GigaChat

Сервис для автоматической расшифровки речи из видеофайлов с использованием нейросети GigaChat от Сбера. Проект помогает создавать текстовые версии видеоуроков, лекций, интервью и другого контента.

## ✨ Возможности

- 📤 Загрузка видео в форматах MP4, MOV, AVI, MKV
- 🎵 Автоматическое извлечение аудиодорожки
- 🤖 Распознавание речи через GigaChat API (поддержка русского и английского языков)
- 📝 Вывод расшифрованного текста с возможностью копирования
- 🖥️ Простой веб-интерфейс с индикацией прогресса
- 🔒 Работа с большими файлами (до 200 МБ видео, до 30 МБ аудио)

## 🛠️ Технологии

| Компонент | Технология |
|-----------|------------|
| Бэкенд | Python 3.10+, FastAPI |
| Распознавание речи | GigaChat API |
| Извлечение аудио | FFmpeg + MoviePy |
| Фронтенд | HTML5, CSS3, JavaScript |
| Сервер | Uvicorn |

## 📦 Установка и запуск

### Требования

- Python 3.10 или выше
- FFmpeg (установленный в системе)
- Аккаунт на [developers.sber.ru](https://developers.sber.ru) с доступом к GigaChat API
