# Inclusive AI — Расшифровка видео

Сервис для расшифровки речи из видео через GigaChat.

## Установка и запуск

### 1. Установите FFmpeg
- **Windows**: [скачайте](https://www.gyan.dev/ffmpeg/builds/), распакуйте в `C:\ffmpeg`, добавьте `C:\ffmpeg\bin` в PATH
- **macOS**: `brew install ffmpeg`
- **Linux**: `sudo apt install ffmpeg`

### 2. Клонируйте проект
```bash
git clone https://gitverse.ru/ваш_логин/inclusive-ai.git
cd inclusive-ai
